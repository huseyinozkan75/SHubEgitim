Css Eğitimi

Css Nedir?

CSS (Cascading Style Sheets - Basamaklı Stil Sayfaları), web sayfalarının görünümünü ve düzenini belirlemek için kullanılan bir stil dilidir. HTML ile birlikte kullanılır ve web sayfalarının tasarımını, renklerini, yazı tiplerini, kenar boşluklarını ve diğer görsel öğelerini kontrol etmek için kullanılır.

Css Yazma Seçenekleri

  1. Inline CSS: Stil kurallarını doğrudan HTML etiketlerinin "style" özniteliği içinde tanımlama.
  2. Sayfa İçi CSS: Stil kurallarını HTML belgesinin style etiketleri içinde tanımlama.
  3. Harici CSS: Stil kurallarını ayrı bir .css dosyasında tanımlama ve HTML belgesine link etme.

Inline CSS Yazımı

Elementlerin açılış etiketlerine style özniteliği eklenerek yazılır. Eğer birden fazla stil kuralı uygulanacaksa, her kural noktalı virgül (;) ile ayrılır. En sondaki kuralın sonunda noktalı virgül kullanılması zorunlu değildir, ancak kodun okunabilirliğini artırmak için önerilir.

Head Tagleri Arasına CSS Yazımı

Sayfa Head alanı içerisinde style etiketleri arasına yazılarak sayfadaki elementlere stil uygulanabilir.

Harici CSS dosyası kullanarak tasarım yapılması

Harici bir CSS dosyası oluşturup (style.css gibi) sayfa head alanı içinde link özelliği ile çağırarak sayfa tasarımını şekillendirebiliriz.

Css Seçiciler

1-Element Seçiciler

Sayfadaki elementleri html taglerine göre (p, h1, div vb) seçerek css uygulayabiliriz.
Bu durumda sayfadaki tüm elementler seçilecektir.

2-Id seçici

Sayfadaki elementleri id özelliğine göre seçip stil uygulamamızı sağlar.
Genelde sayfadaki tek bir elementi seçmek için kullanırız.
Css dosyası içerisinden # karakteri yazılıp elementin Id değeri eklenerek element seçilir
Örnek: #idSeçici{}

3-Class Sınıf Seçici

Web sayfasında veya uygulama genelinde birden fazla elemente aynı stil ayarlarını uygulamak için class seçicileri kullanırız.

Element tagi içerisine class yazıp css dosyasında yazılan class adını vererek kullanırız.

Css dosyası içerisinde . işaretiyle class oluşturuyoruz.
Örneğin: .sinifSecici{}

Bir elemente class özelliğiyle içerisine aralarda boşluk bırakarak birden fazla class uygulayabiliriz.

Ana div içindeki ilk p

Bir html elementin içindeki alt elementleri de hedefleyerek stil uygulayabiliriz. Örneğin: #anaDiv p{}
