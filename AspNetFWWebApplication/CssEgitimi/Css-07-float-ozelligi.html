Css Float Kullanımı

Css 'de "float" özelliği, bir öğeyi belge akışından çıkararak sola veya sağa hizalamak için kullanılır.
float: left; özelliği verilen div sola yapışır. içeriği kadar yer kaplar, sonrasında gelen element buradan hizalanarak akışa devam eder.
float: left verilen elementten sonraki element.
float: left verilen elementten sonraki 2. element.
float: left verilen elementten sonraki 3. element.

Float-Clear

Float özelliğini kaldırmamız gerektiğinde "clear" özelliğini kullanırız.

Float Left Div i
float:left den sonraki div

float özelliği verilen bir elementten sonraki elementi yapışma olayından kurtarmak için clear: left-right-both kullanılır.
clear:left: Sadece soldaki float elementten kurtarır.
clear:right: Sadece sağdaki float elementten kurtarır.
clear:both: Hem sağdaki hem soldaki float elementten kurtarır.
