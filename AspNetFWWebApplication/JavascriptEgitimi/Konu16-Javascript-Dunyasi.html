Javascript ile neler yapılabilir

1. Web (Frontend) Uygulamaları

Kullanılan popüler kütüphaneler / framework'ler

React.js Vue.js Angular Svelte

Bu kütüphaneleri kullanarak SPA (Single Page Application) geliştirme yapabiliriz.

2. Sunucu Tarafı (Backend) Uygulamaları

Javascript, Node.js sayesinde sunucu tarafında da çalışabilir.

Örnek teknolojiler:

Node.js, Express.js, NestJS, Fastify

3. Mobil Uygulamalar

Javascript ile native mobil uygulamalar da geliştirilebilir

Kullanılan Araçlar:

React Native => iOS ve Android uygulamaları için,
Ionic veya Cordova => web tabanlı mobil uygulamalar
NativeScript

4. Masaüstü Uygulamaları

Javascript, HTML ve CSS ile masaüstü uygulamaları da oluşturulabilir.

Kullanılan Araçlar

Electron.js => VS Code, Slack gibi uygulamalar bu teknolojiyle yapılmıştır.
Tauri => Electron'a alternatif, daha hafif bir çözüm

5. Oyun Geliştirme

JavaScript, özellikle 2D tarayıcı tabanlı oyunlar için uygundur.

Kullanılan Araçlar

Phaser.js
Three.js (3D oyunlar için)
Babylon.js

6. Makine Öğrenmesi, IoT ve Diğer Alanlar

  • Makine Öğrenmesi
  • Tensorflow.js ile model eğitimi ve tahminler
  • IoT (Nesnelerin İnterneti)
